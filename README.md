# LectioWeb

Site web statique pour la conformité Google de l'application Lectio.

## Pages disponibles

- `index.html` : page d'accueil de conformité
- `privacy.html` : politique de confidentialité
- `terms.html` : conditions d'utilisation

## Publication avec GitHub Pages

Le workflow GitHub Actions `.github/workflows/pages.yml` déploie automatiquement le site sur GitHub Pages à chaque push sur `main`.

URL attendue après activation de GitHub Pages :

`https://amande-andre.github.io/LectioWeb/`
